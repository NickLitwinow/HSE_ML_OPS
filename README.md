# Credit Default Risk Prediction Service

**Команда:** [Ваше Имя]

## Описание проекта
Проект предназначен для оценки кредитного риска (вероятности дефолта) заявителей на получение кредита. Сервис помогает автоматизировать принятие решений о выдаче кредитов, снижая финансовые риски банка.

Решение разрабатывается с соблюдением принципов MLOps для обеспечения надежности, воспроизводимости и автоматизации.

## Основные возможности
- **Предсказание:** Оценка вероятности дефолта на основе анкетных данных и кредитной истории.
- **API:** REST API для интеграции с банковскими системами.
- **Мониторинг:** Отслеживание качества данных и модели в продакшене.

## MLOps Практики
В проекте реализован полный цикл MLOps:
1.  **Управление кодом:** Git flow, пре-коммит хуки.
2.  **Управление данными:** DVC (Data Version Control) для версионирования датасетов.
3.  **Эксперименты:** MLflow для трекинга метрик и параметров моделей.
4.  **Контейнеризация:** Docker для изоляции окружения сервиса.
5.  **CI/CD:** GitHub Actions для автоматического тестирования, линтинга и сборки образов.
6.  **Инфраструктура:** Docker Compose для локального развертывания.

## Структура репозитория
- `src/`: Исходный код сервиса и пайплайнов обучения.
- `notebooks/`: Jupyter ноутбуки для EDA и прототипирования.
- `tests/`: Модульные и интеграционные тесты.
- `.github/`: Конфигурации CI/CD.
- `data/`: Локальное хранилище данных (под управлением DVC).
